# Плагин документации Dock Rush

Dock Rush — это оболочка для документации на React, которая превращает папку
markdown‑файлов в полноценный сайт документации.

Возможности:

- несколько версий документации;
- поддержка разных языков;
- группы и выпадающие разделы в сайдбаре;
- кнопки-ссылки и кнопки, открывающие страницы;
- полнотекстовый поиск по заголовкам и содержимому.
